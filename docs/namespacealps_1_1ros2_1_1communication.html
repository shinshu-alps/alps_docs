CAN通信データからROSトピックデータに変換する

変換可能な型を追加したい場合はこのテンプレート関数の特殊化を定義する．
デフォルトで用意されている変換は以下の通り (intNに入るのは8, 16, 32, 64) .

  • bool -> std_msgs::msg::Bool
  • char -> std_msgs::msg::Char
  • uint8_t -> std_msgs::msg::Byte
  • intN_t -> std_msgs::msg::IntN
  • uintN_t -> std_msgs::msg::UIntN
  • float -> std_msgs::msg::Float32
  • double -> std_msgs::msg::Float64
  • std::array<uint8_t, kSize> -> std_msgs::msg::ByteMultiArray
  • std::array<intN_t, kSize> -> std_msgs::msg::IntNMultiArray
  • std::array<uintN_t, kSize> -> std_msgs::msg::UIntNMultiArray
  • std::array<float, kSize> -> std_msgs::msg::Float32MultiArray
  • std::array<double, kSize> -> std_msgs::msg::Float64MultiArray
  • etl::vector<uint8_t, kMaxSize> -> std_msgs::msg::ByteMultiArray
  • etl::vector<intN_t, kMaxSize> -> std_msgs::msg::IntNMultiArray
  • etl::vector<uintN_t, kMaxSize> -> std_msgs::msg::UIntNMultiArray
  • etl::vector<float, kMaxSize> -> std_msgs::msg::Float32MultiArray
  • etl::vector<double, kMaxSize> -> std_msgs::msg::Float64MultiArray
  • etl::string<kMaxSize> -> std_msgs::msg::String
